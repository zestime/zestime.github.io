뜻하지 않게, 맥북을 새로 설치하게 되었습니다. 모두들 저마다의 설정법이 있겠지만, 저는 이런 식으로 한답니다.

TRIM ON 트림 기능 활성화

(SSD 유저에 한함) 듣기로는 서드 파티의 SSD에 대해서는 TRIM이 비활성화 되어 있습니다. 이는 System Information에서 확인 가능합니다. 저는 OCZ Agility 3를 사용하는데, 역시나 비활성화 되어있습니다. SSD가 사용하는 꼭 필요한 기능이므로(자세한 설명은 http://en.wikipedia.org/wiki/TRIM) Trim Enabler를 이용해서, 해당 기능을 활성화 시킵니다. Trim Enabler는 http://www.groths.org/trim-enabler/ 의 경로에서 받을 실 수 있습니다.

설치 이후에, 응용 프로그램을 실행시키면, AppStore 출신이 아니라고 실행되지 않습니다. 이는 [System Preferences] - [Security & Privacy] 에서 변경하실 수 있습니다. 하단의 자물쇠를 풀고, Allow Application downloaded from 항목을 Anywhere로 변경하시면 됩니다.

전원 설정도 바꾸어야 하는데, SSD의 경우에는 아시다싶이 전원 끄기 기능을 수행하면 안됩니다. 전원 설정 화면에서, 'Put hard disks to sleep when possible'을 비활성화 합니다.

Keyboard 키보드 설정

키보드에 한글을 추가합니다. 저는 영문으로 설정해서 사용하는 관계로, 한글을 추가합니다. 이는 [System Preference] - [Keyboard]에서 Input Sources를 이용해서 추가하면 됩니다. 간단해요.

그리고 단축키(Shortcut)을 설정해야 하는데, 일반적으로 Space + Command 키로 많이 설정합니다. 그런데, Spotlight에 설정되어 있으므로, 해제하시고 할당해 주시면 됩니다.

맥에서는 F1~F12키가 펑션키로 할당되어 있습니다. 만약, F10키를 누르고 싶다면, Fn + F10을 누르셔야, F10가 동일한 효과가 나타납니다. 펑션키를 자주 사용하지 않고, F1~F12를 더 많이 사용하신다면, 'Use all F1, F2, etc. keys as standard function keys'에 체크하시면 됩니다.

Change Home directory 홈 폴더 변경하기

저는 따로 Time Machine을 쓰지 않고, Home 폴더를 다른 디스크에 저장하여 사용합니다. 부득이하게 OS를 재설치하더라도, 제가 이전에 쓰던 파일들은 다른 영역에 저장되어 있기 때문에, 영향을 주지 않습니다. 백업을 한 건 아닌데, 다른 영역이라 서로 영향을 미치지 않는 것 입니다. 이렇게 하기 위해서는, OSX가 자동적으로 제공하는 Home 폴더가 아닌, 다른 폴더를 설정하셔야 합니다.

  1. 이는 [System Preferences] - [Users & Groups] 에서 설정하실 수 있습니다.
  2. 하단의 자물쇠를 클릭하여, 패스워드를 입력하시면 자물쇠가 풀린 상태가 됩니다.
  3. 좌측 리스트에서 유저에 커서를 위치하고 우클릭을 하시면, 'Advanced Options'을 클릭합니다.
  4. 유저에 대한 상세한 정보가 보여지는데, 항목 중에서 'Home directory'를 위치하고자 하는 파티션의 임의의 폴더로 설정합니다.
  5. 만약, 임의의 폴더가 없는 상태라면, 빈 디스크(혹은 파티션)에 Users 폴더를 통째로 복사하여 설정하시면 됩니다.

Recommended Programs 편의 프로그램 설치

  • Quicksilver - 일종의 Launcher입니다만, 대부분의 프로그램 실행을 간편하게 해 줍니다.
  • MenuMeters - 메뉴바에 CPU, Network Traffic 등을 표시해주는 프로그램입니다.
  • Chrome - 기호의 문제이지만, Bookmark sharing 때문에 넘어왔습니다.
  • Evernote - 저는 에버노트로 메모 관리를 합니다. AppStore에서 받으시면 됩니다.
  • ClipMenu - Copy & Paste를 편하게 해주는, ClipBoard 프로그램입니다.
  • iTerm2 - Mac에 기본적으로 제공되는 Terminal은 너무 구려서;;;
  • VirtualBox - 저는 페렐렐즈 혹은 부트캠프를 사용하지 않습니다. 윈도우가 필요한 경우에는 VirtualBox로 가상 머신으로 사용합니다. 성능 저하가 없는 것은 아니지만, 저의 경우에는 충분했습니다.(게임을 하는 경우가 아니라면, 굳이 윈도우를 Native로 사용할 필요가 있을지 모르겠네요. 저는 개발을 했는데, 약간 느린 정도 였습니다.)
