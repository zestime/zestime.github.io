음.. 간단히 말씀드리면, 윈도우에서 Window + R 키를 조합하여 사용하던 실행창의 역활을 하는 프로그램입니다. 물론, LaunchPad에서도 Searching이 가능하기에, 단순히 LanchPad에서 Application의 이름을 입력하여 사용해도 됩니다. 다만, 조금 더 확장된 기능을 제공하기에, Quicksilver를 사용합니다.

Fingertip이라고 부르는데요, 프로그램을 실행시키는 일종의 Launch(런쳐)입니다. 다른 점은, 키보드로 간단하게 실행시킬 수 있다는 점입니다. 이러한 종류의 프로그램은Quicksilver외에도 Alfred도 있습니다. 기호나 개인의 차이이니, 살펴보시고 편하신 프로그램을 사용하시면 되겠습니다. 저는 Quicksilver를 더 선호하는데, 이는 Quicksilver의 작은 학습 기능과 플러그인 때문입니다.

Install 설치

Quicksilver 사이트에서 내려 받으시면 됩니다.

 

 

 

 

 

 

 

 

다운받은 dmg파일을 마운트하셔서, Application(응용 프로그램) 폴더로 드래그하시면 되겠습니다.

Run 실행

실행시켜보면, 역시나 보안에 대한 안내가 나타납니다.

가볍게 'Open'을 하시면,

첫 실행 화면이 나타납니다. Continue(계속)을 통해, 넘기다 보면 Plug-in(플러그인) 설정 화면이 나타납니다. Quicksilver의 강력한 기능 중에 하나는, 바로 이 Plug-in입니다.

자주 사용하는 Application의 Plugin을 설치하시면 다음부터는 쉽게 사용이 가능하게 됩니다. 예를 들면, 아이튠즈에서 음악을 들을 때, Quicksilver의 fingertip에서 간단하게 그 노래 제목으로 실행시키는 형태입니다. 아무래도, 한 번에 실행시키게 되니까 더욱 빠르고 간편하겠죠.

Preference 설정

아시다 싶이, Mac에서 모든 설정은 ⌘, 키입니다. Quicksilver도 예외가 아닙니다.

여기서, 항상 컴퓨터를 켤 때마다 자동 실행되도록 Start at login에 체크를 합니다. 그리고, 항상 실행시킬 것이기 때문에 굳이 dock에 보여질 필요는 없을 것 같네요. 따라서, Show icon in dock은 해제합니다. 보시다 싶이, 설치하면서 설정한 내용들은 모두 여기에서 변경 가능합니다.

Application

앞서 말씀드린 것처럼, Start at login을 체크를 하고 나머지는 빼버렸습니다. 물론, 이런 상태에서는 Dock에 하나의 Application으로 나타나지 않기 때문에, 메뉴나 '⌘Tab' 으로 접근하기 어렵습니다. 이런 경우에는 fingertip에서 제공되는 버튼을 활용하시면 되겠습니다.

Command

앞서 말씀드린 것 처럼, spotlight와 보완적인 관계로 사용하실 수 있습니다. spotlight는 컴퓨터내의 모든 파일에 대한 정보를 가지고 있습니다. 반면에, Quicksilver는 Application에 대한 정보를 가지고 있는 셈입니다. 따라서, 프로그램을 실행시키는 경우에는 Quicksilver로 간단하게 실행시키고, 문서나 사진을 찾는 경우에는 spotlight를 사용하게 됩니다.

위에서 보시는 것처럼,

  • Alt + Space - Quicksilver의 단축키
  • Ctrl + Space - Spotlight의 단축키

로 설정해서, 같이 사용할 수 있도록 해 두었습니다. 사용해 본 결과, 대부분은 Quicksilver로 충분하기에, Spotlight는 정말 가끔 사용하게 됩니다. 그래도, OS에서 제공하는 편의 기능이니까, 잘 기억해두는 것이 좋습니다.

'When activated, switch keyboard to U.S.'라는 옵션에 체크해 두었습니다. 이는, 한글을 번갈아 사용하는 경우에 Quicksilver에서 영문을 우선 적용하겠다는 옵션입니다. 사용하시는 프로그램의 이름이 대부분 영문이라면, 굳이 한글로 프로그램을 찾을 이유가 없을 것 입니다. 실수를 방지하기 위해서, 해당 옵션을 체크하시면 더욱 편한 Mac Life를 즐길 수 있습니다.
