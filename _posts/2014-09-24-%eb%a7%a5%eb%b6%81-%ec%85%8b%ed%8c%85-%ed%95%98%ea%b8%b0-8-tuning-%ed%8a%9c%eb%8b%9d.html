악기만큼은 아니지만, 컴퓨터도 Tuning(튜닝)이 필요합니다. 튜닝이라기 보다는 자잘한 성능 개선 팁에 가깝지만, 그래도 안하는 것 보다야 당연히 좋겠죠. Mac은 상당히 좋은 OS이지만, 모든 인간의 작품이 그러하듯 군더더기가 있습니다. 조금씩 개선해 보도록 하죠.

Dashboard 대쉬보드 끄기

대쉬보드는 여러 위젯을 등록해 놓고 간편하게 사용할 수 있는 좋은 기능입니다. 그렇지만, 저는 사용하지 않아요. 저처럼, 대쉬보드가 필요 없으신 분들은 깔끔하게 지워버리느게 좋겠죠. Terminal(터미널)을 실행시켜서, 다음을 입력합니다.

defaults write com.apple.dashboard mcx-disabled -boolean YES
killall Dock

물론, 복원하는 법도 알고 계셔야겠죠. 정말, 써 본 적은 없지만, 다음과 같다고 하네요.

defaults write com.apple.dashboard mcx-disabled -boolean YES
killall Dock

 

 Swap File 스왑 파일 해제

MenuMeter를 켜 놓고, 맥을 쓰다 보면 생각보다 메모리를 다 사용하지 않는 것을 알 수 있습니다. 노트북의 메모리가 8G인데, 풀로 다 쓰는 경우는 없었습니다. 제가 그래픽 작업을 하지는 않아서 인지, 늘 조금씩 남아 있었습니다. 이런 경우에 굳이 Swap이 필요 없기 때문에, 해당 기능을 끄도록 하겠습니다.

[caption id="attachment_592" align="alignnone" width="389"] MenuMeter에서 확인한 Memory Status[/caption]

현재, 이렇게 1G의 Swap을 사용하고 있습니다. 필요없기 때문에, Swap 파일을 삭제하고 Swap 기능도 끄도록 하겠습니다.

# Swap 기능 끄기 
sudo launchctl unload -w /System/Library/LaunchDaemons/com.apple.dynamic_pager.plist
# Swap 파일 삭제
sudo rm /private/var/vm/swapfile*

해당 기능을 켤 수도 있습니다. 다음의 명령어로 필요하실 때, 켤 수 있습니다.

# Swap 기능 켜기
sudo launchctl load -wF /System/Library/LaunchDaemons/com.apple.dynamic_pager.plist

간단하지만, 편리하게 사용할 수 있습니다.
