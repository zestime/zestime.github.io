Meter라는 표현이 어색하실 수도 있을 것 같습니다. 모니터링을 하는 Tool인데, 주로 CPU, Memory, Network Throughput(전송량) 등을 표시하여 줍니다. 윈도우에서는 주로 위젯으로 많이 사용되는데요, 저는 Menubar에 위치하여, 언제든 확인할 수 있도록 사용하고 있습니다. 이렇게 제공되는 여러 프로그램이 있지만, 무료라는 대원칙을 지키는 MenuMeters를 소개합니다.

MenuMeters

Install 설치하기

MenuMeters 사이트에서 내려 받으시면 됩니다. 다행히도, Max OS X10.4 이후 버전은 모두 지원되고 있습니다. 받으신 dmg파일을 마운트하셔서 Installer를 실행시킵니다.

특별히, 특정 사용자만 사용할 것이 아니라면, all users를 선택합니다.

설치가 완료되면, 아무런 메세지도 별다른 창이 뜨지는 않습니다. 처음에는 저도 어리둥절했습니다. 어떻게 사용해야 하는거지?

Setting 설정하기

설치하신 MenuMeters는 응용 프로그램이 아니기 때문에, LaunchPad에서는 확인되지 않습니다. System Preferences(환경 설정)에 들어가 보면, 새롭게 추가된 항목을 보실 수 있습니다.

제일 하단에 추가된 MenuMeters를 선택합니다.

보시는 것 처럼, 4가지 탭으로 구성되어 있습니다. 당연한 이야기겠지만, 각 항목별로 사용자가 선택하여 표시되는 방식으로 되어 있습니다. CPU의 경우만 보더라도, Interval(측정 시간 간격)과 표시 방식 및 색상을 결정할 수 있습니다. 저는 다음과 설정했습니다. 코어 수 만큼, 온도계(Thermometer) 모양으로 표시하는 설정입니다.

Disk는 특별히 모니터링 하지 않기에, 아무런 설정을 하지 않았습니다.

Memory의 경우에는 원 차트로 보는 걸 좋아합니다. 특별히 변경할 것은 없어서, 단순히 켜기만 했습니다.

Network의 경우는, 화살표로 표시되는 것 보다, 자리는 조금 많이 차지해도 현재 데이타 전송량을 보여주는 것이 가끔 편해서 Throuhput(처리량)으로 표시해 놓고 사용합니다.

결과적으로 메뉴바는 다음과 같이 구성되었습니다. 항목간의 순서를 부여하는 메뉴는 없지만, Display Meter라는 체크가 먼저 된 항목이 오른쪾에 위치하게 됩니다. 저의 경우에는, Network를 제일 먼저 활성화시켰기에, 우측에 보여지고 있습니다.

 

간단한 유틸이지만, 생각보다 Mac의 상황을 파악하는데 편리합니다. 각 아이콘을 클릭하여 보시면, 관련된 자세한 정보를 확인할 수 있습니다. 여러분들도 조금 더 편리한 Mac Life를 즐기시기 바랍니다.
